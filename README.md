# 手势识别系统 python219

## 项目概述

手势识别系统是基于 Python 和 OpenCV 开发的应用，主要用于识别动态手势，实现数字 1-10 的手势识别功能。

## 技术栈

- **后端：**
  - Python
  - OpenCV

## 功能模块

- **手势识别核心：**
  - 识别动态手势
  - 数字 1-10 手势识别

## 系统角色

- 用户：通过摄像头捕捉手势，系统识别后展示相应数字。

## 运行环境

- 操作系统：兼容 Windows、macOS、Linux
- Python 环境要求：Python 3.x，已安装 OpenCV 库

## 部署步骤

1. 确保计算机已安装 Python 3.x 和 OpenCV 库
2. 克隆或下载项目代码
3. 在项目目录下运行主程序

## 目录结构

```
手势识别系统/
├── main.py          # 主程序
├── gesture_module/  # 手势识别模块
└── utils/           # 工具库
```

## 核心亮点

- **实时性：** 系统可实时捕捉并识别手势，响应速度快。
- **准确性：** 基于OpenCV强大的图像处理能力，能准确识别数字 1-10 的手势。
- **兼容性：** 跨平台运行，满足不同用户需求。
